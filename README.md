# 投资复利计算器

一个完全网页端的投资复利计算工具，帮助用户根据投资本金、投资年限、定投金额和预计年化收益率，快速查看资产增长轨迹、累计收益和年度明细。

项目采用原生 `HTML + CSS + JavaScript` 实现，不依赖后端，也不需要构建工具，打开即可使用。

## 功能特性

- 支持输入投资本金、投资年限、单次定投金额、预计年化收益率
- 支持定投频率切换：`按月` / `按年`
- 支持定投时点切换：`期初投入` / `期末投入`
- 支持两种计算模式：
  - `固定年化预测`
  - `历史回测`
- 自动计算：
  - 最终总资产
  - 累计本金
  - 累计收益
  - 累计收益率
- 支持 `目标反推`：
  - 输入目标总资产
  - 自动反推在当前假设下所需的单次定投金额
  - 展示与当前计划之间的差额
- 提供双图表视图：
  - `资产拆解`：总资产 / 累计本金 / 累计收益
  - `对比视图`：
    - 固定预测模式下显示 `保守 / 基准 / 乐观`
    - 历史回测模式下显示 `组合累计收益率 / 标的累计涨幅`
- 支持历史回测：
  - 内置 `标普 500`
  - 内置 `纳斯达克 100`
  - 内置 `沪深 300`
  - 支持回测起点切换
- 提供 `结果解释区`，自动总结目标达成度、复利接棒年份、收益结构与样本特征
- 鼠标悬停图表时显示当前年份详细数值
- 提供年度明细表，展示每年的资产变化
- 支持导出年度明细为 `CSV`
- 适配桌面端与移动端浏览

## 计算模式

### 1. 固定年化预测

在这个模式下，系统使用用户输入的年化收益率进行长期复利模拟，并生成保守、基准、乐观三种情景。

### 2. 历史回测

在这个模式下，系统使用内置的历史年度收益样本，按所选起点逐年回放资产路径。

当前内置回测标的包括：

- `标普 500`
- `纳斯达克 100`
- `沪深 300`

## 计算逻辑

本项目采用固定年化收益率模型，并将年化收益率换算为等效月收益率后进行逐月模拟。

### 月收益率换算

```text
月收益率 = (1 + 年化收益率)^(1/12) - 1
```

### 模拟原则

- 初始本金在第 0 年计入资产
- 每个月按当前资产计算收益
- 定投可发生在每期开始或每期结束
- 若选择按年定投，则在每年对应时点投入一次
- 每年汇总一次，生成年度明细

### 情景对比

系统会基于基准年化收益率和“情景波动区间”生成三种预测曲线：

- 保守情景 = 基准收益率 - 波动区间
- 基准情景 = 用户输入收益率
- 乐观情景 = 基准收益率 + 波动区间

### 目标反推

目标反推会在当前投资期限、当前模式和当前参数约束下，反向求解达到目标终值所需的单次定投金额。

如果当前计划已经超过目标，则会显示理论上可以降低到的单次投入金额。

## 结果说明

页面会展示以下关键指标：

- `最终总资产`：投资结束时的账户总值
- `累计本金`：初始本金 + 所有定投投入
- `累计收益`：总资产减去累计本金后的净收益
- `累计收益率`：累计收益 / 累计本金

目标反推模块会额外展示：

- `目标终值`
- `所需单次定投`
- `当前计划差额`

结果解释区会自动生成：

- 目标达成度
- 复利接棒年份
- 收益结构拆解
- 固定预测或历史回测下的补充洞察

年度明细表则会进一步展示：

- 年初资产
- 本年投入
- 累计本金
- 本年收益
- 年度回报
- 累计收益
- 年末总资产
- 累计收益率

## 适用场景

这个工具适合用来：

- 快速估算长期投资的复利效果
- 比较不同收益率假设下的资产增长差异
- 反推达到目标资产所需的定投金额
- 用历史样本观察不同市场路径下的长期投入体验
- 演示定投策略对长期资产积累的影响
- 作为投资教育或金融工具类前端作品集项目

## 注意事项

当前版本同时包含固定预测和历史回测两种模式，适合做静态估算、产品演示和投资教育，不代表真实投资结果。

以下因素暂未纳入计算：

- 手续费
- 管理费
- 税费
- 通货膨胀
- 市场波动与最大回撤
